# Universal File Analyzer

Universal File Analyzer is a versatile tool that allows you to analyze various types of files quickly and efficiently. Whether it's text, CSV, or other common file formats, this application helps you inspect, explore, and extract useful information from your files with ease.

## Requirements

- Python 3.x
- Streamlit
- python-dotenv
- groq

## Installation

Run the following commands to install the required packages:

python -m pip install streamlit
python -m pip install python-dotenv
python -m pip install groq

## How to Run

python -m streamlit run src\universal_file_analyzer_app.py
